Goal

The goal of this project is to predict a male workerâ€™s wage from the Mid-Atlantic region based on his age, marital status, race, education, job class, and year that wage information was recorded.

Data

The Wage data was obtained from the Introduction to Statistical Learning with Applications in R (ISLR) package in R. It contains 3,000 observations of 11 variables.

Analysis

After a response variable and one or more predictor variables are chosen from the menus on the left, each tab performs the following functions:

  • Data Summary: generates 5-number summaries for each numeric variable and frequencies for each categorical variable
  • Exploratory Plots: shows univariate plots for each variable
  • Variable Selection: performs 10-fold cross-validation on the model designated by the user and plots the mean cross-validation error as a function of model size; allows the user to select the number of variables to include in the final model based on the plot; shows the coefficients of the best subsets model for the chosen number of variables
  • Confidence Intervals: calculates 95% bootstrap confidence intervals for each coefficient of the model obtained after performing variable selection
  • Prediction Interval: calculates a 95% prediction interval for a new observation, which is designated as the median of each predictor (performed after variable selection and confidence intervals)
